Saskatoon 500 Car Race

Object-oriented JavaScript racing simulation with ES6 classes, animation intervals, and persistent statistics.

Start Racing

Summary

This project involved creating an animated racing game using object-oriented JavaScript with ES6 classes. Four cars race across the screen using random movement, with the first to cross the finish line declared the winner.

The implementation demonstrates key OOP concepts including class constructors, encapsulation, methods, and state management. The game uses setInterval() for smooth animation and maintains persistent statistics across multiple races.

Role & Technologies

Individual Project HTML5 CSS3 JavaScript ES6 OOP Classes 2 Weeks

Key Features

  • Object-oriented design with Car and Race classes
  • Random movement generation (1-10 pixels per frame)
  • Real-time position tracking with absolute positioning
  • Automatic finish line detection at 1000px
  • Race standings tracking in finish order
  • Persistent win statistics across multiple races
  • Button state management to prevent race conflicts
  • Clean reset functionality for multiple races

Technical Highlights

Object-Oriented Architecture

Car Class: Encapsulates individual car properties including position tracking, rendering with inline styles, random movement generation, and win count persistence.

Race Class: Manages the overall race state including the array of cars, race completion status, standings tracking, and race statistics.

Animation and Timing

The game uses setInterval() with a 10-millisecond delay to create smooth 100 FPS animation. On each tick, all unfinished cars move a random distance (1-10 pixels), creating unpredictable racing dynamics.

How to Race: Click "Start Race!" to begin. Cars will race across the screen with random movement speeds. The first car to reach the finish line wins! Use "Reset Race" to race again, and "View Statistics" to see which car has won the most races.
